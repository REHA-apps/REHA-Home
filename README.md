# REHA Apps Homepage

Professional landing page for **REHA Apps**, showcasing smart marketplace integrations for HubSpot, Slack, WhatsApp, and more.

## ✨ Features

- **Modern UI/UX**: Built with a sleek dark theme (Charcoal/Navy) and Teal branding.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile viewports.
- **Symmetric Navbar**: Clean, centered branding layout for a premium feel.
- **Interactive Components**:
  - Live stats counter for integrations and messages.
  - Glassmorphic integration cards with hover effects.
  - Animated hero section with gradient orbs.
  - Automated scroll-reveal animations.
- **Marketplace Ready**: Includes official high-quality SVGs for HubSpot, Slack, WhatsApp, and Microsoft Teams.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure.
- **Tailwind CSS v3**: Utility-first styling via CDN.
- **Vanilla JavaScript**: Lightweight animations and DOM interactions.
- **Google Fonts**: Inter font family for modern typography.

## 🚀 Deployment

The website is a static project and can be hosted on any platform like **GitHub Pages**, **Netlify**, or **Vercel**.

### Local Preview
Simply open `index.html` in any modern web browser.

### Hosting on GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/ (root)` folder.
4. Click **Save**.
